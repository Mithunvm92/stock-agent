# 🤖 Stock Trading Agent

Self-trainable ML-based trading agent with capital protection.

## Features
- Self-training ML model (retrains daily)
- 28 technical indicators
- 6-layer risk management
- Paper trading mode
- Backtesting engine
- Multi-stock scanning
- Trade logging
